# Cpplings — Module 01: Foundations

This folder contains Rustlings-style C++ practice exercises for Module 01.

## Recommended compiler command

```bash
g++ -std=c++23 -Wall -Wextra -pedantic exercise.cpp -o exercise
```

For multi-file exercises:

```bash
g++ -std=c++23 -Wall -Wextra -pedantic main.cpp other.cpp -o program
```

## Structure

```text
cpplings_module_01/
├── README.md
├── exercises.md
├── answer_key.md
├── starter/
│   └── starter files for each exercise
└── solutions/
    └── solved versions where useful
```

## Suggested workflow

1. Open `exercises.md`.
2. Work through each exercise in order.
3. Use the files in `starter/` as your broken/incomplete code.
4. Check `answer_key.md` only after attempting the exercise.
